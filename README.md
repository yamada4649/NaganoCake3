# ケーキ通販サイト

ケーキを注文できるアプリケーションとなっております。</br>
実装内容として、顧客のログイン機能、商品登録、カート機能、注文機能の実装を行っています。</br>
このアプリケーションのポイントとして、複数の商品写真を登録でき、ジャンルから商品を選択することができます。</br>
また、topページのレイアウトに関して、高級感を出すために、背景を黒色にしています。


アプリケーションの実装機能について:

* deviseによるログイン機能

* refileによる画像機能

* bootstrapによるレイアウト補正

* カート機能の実装

* 注文機能の実装

* namespaceによる顧客側と管理者側の分岐

* アソシエーション関係の実装

* 商品のいいね実装
